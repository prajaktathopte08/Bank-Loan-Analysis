# 📊 Bank Loan Analysis (SQL + Power BI)

## 🔍 Project Overview
This project focuses on analyzing bank loan data to understand loan performance, customer behavior, and risk factors.  
The goal is to identify **Good Loans** and **Bad Loans** and provide insights for better decision-making.


## 🎯 Objectives
- Analyze loan performance and trends  
- Identify good vs bad loans  
- Understand customer behavior  
- Provide data-driven insights  


## 🛠️ Tools & Technologies
- **SQL** – Data cleaning & analysis  
- **Power BI** – Data visualization & dashboard  
- **Excel/CSV** – Data source  


## 🧹 Data Cleaning (SQL)
- Fixed incorrect data types  
- Handled missing values  
- Resolved column misalignment  
- Standardized `loan_status` values  



## 📊 Dashboard Features

### 🔹 KPI Metrics
- Total Loan Applications  
- Total Funded Amount  
- Total Amount Received  
- Average Interest Rate  
- Average DTI  


### 🔹 Loan Analysis
- Good Loan % (Fully Paid + Current)  
- Bad Loan % (Charged Off)  


### 🔹 Visual Insights
- Monthly Loan Trends 📈  
- State-wise Analysis 🗺️  
- Loan Purpose Distribution 📊  
- Home Ownership Analysis 🏠  
- Employment Length Analysis 👨‍💼  


## 🎛️ Interactive Features
- Slicers (State, Grade, Purpose)  
- Dynamic filtering  
- Parameter-based measure selection  


## 📈 Key Insights
- ✔ ~86% loans are Good Loans  
- ✔ ~14% loans are Bad Loans  
- ✔ Loan applications show increasing trend  
- ✔ Debt consolidation is the most common purpose  
- ✔ Customers with lower DTI are less risky  


## 💡 Business Impact
- Helps identify high-risk loans  
- Improves loan approval strategy  
- Supports data-driven decision making
